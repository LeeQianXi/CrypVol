## 加密实现说明

CrypVol 使用以下加密原语：

- **密钥派生**: Argon2id
- **对称加密**: AES-256-GCM（CEK 加密）/ AES-256-CBC（数据加密）
- **非对称加密**: RSA / ECC（用于 CEK 传输）

如果你发现任何加密实现中的弱点，请通过上述渠道报告。

## 最佳实践

- 永远不要将 `.cvk` 密钥文件与 `.cvp` 数据卷放在同一位置
- 使用 `Password` 或 `Asymmetric` 模式时，确保密码 / 私钥强度足够
- 定期轮换加密密钥
